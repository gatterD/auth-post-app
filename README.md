# Auth Post App

Веб-приложение для просмотра пользователей и постов с демонстрацией механизма авторизации через токен GoRest API.

**Демо:** [https://candid-druid-706e55.netlify.app/](https://candid-druid-706e55.netlify.app/)

## Описание

Проект представляет собой одностраничное приложение (SPA), разработанное в качестве тестового задания. Приложение взаимодействует с публичным API [GoRest](https://gorest.co.in/) для получения и отображения данных о пользователях и их постах.

Ключевая особенность: для доступа к API требуется Access Token, который пользователь вводит при первом запуске. Токен сохраняется в `localStorage` с помощью библиотеки `zustand`, что позволяет не вводить его повторно при последующих посещениях.

### Функциональные страницы

- **Главная страница** – приветственный экран и форма для ввода токена.
- **Пользователи** – список пользователей с постраничной навигацией.
- **Посты** – список всех постов с постраничной навигацией.
- **Детальная страница пользователя** – информация о пользователе и список его постов.
- **Детальная страница поста** – полный текст поста.

### Технологический стек

- **Frontend:** React 18, TypeScript
- **Сборка:** Vite
- **Маршрутизация:** React Router v6
- **UI-библиотека:** Consta UI Kit
- **Управление состоянием:** Zustand (для хранения токена)
- **HTTP-клиент:** Axios
- **Деплой:** Netlify

## Запуск проекта локально

### Требования

- Node.js (версия 18.x или выше)
- npm

### Инструкция по установке

1. Клонирование репозитория:
   ```bash
   git clone https://github.com/gatterD/auth-post-app.git
   cd auth-post-app
2. Установка зависимостей:
    npm install
3. Запуск в режиме разработки:
    npm run dev
4. Открыть приложение в браузере по адресу http://localhost:3000
